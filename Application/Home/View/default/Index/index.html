OPENED,就是按满意度付费

      “按满意度付费”在广告业可是个新标准，不过别担心，它一点都不难理解。

      以往，收费的多少要在合作之前，通过谈判决定，双方的专家预估工作量。企业的支付能力，广告公司的水平等等一些因素来确定金额。这样做耗费了大量的精力，最头疼的是：预估和实际往往差异非常大，比如遇到突发问题需要额外的时间解决，或是广告公司发挥的不好等等。这造成了广告行业的诸多问题，但其实这些问题都能避免，方法就是--后计费。



      OPENED “按满意度付费” 每小时150元服务费，根据实际用了多少小时，结合客户的评价来付费。

      许许多多的中小企业客户选择OPENED，就是因为这种付费方式省心，透明而且很公平：合作前不用在方法或是成本上兜圈子，工作中坦诚认真，专注于结果。看上去是一点点心态的小改变，却能激发友好的氛围，使广告服务更高效。

      如果你也喜欢公平的合作方式，说不定这也是你正在找的。
